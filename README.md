# ❤️✨ Tic-Tac-Toe: Hearts & Stars ✨⭐

A fun and interactive twist on the classic Tic-Tac-Toe game, built as part of my **Web Development Internship at Prodigy Infotech**. This version adds playful **Hearts ❤️ and Stars ⭐ icons**, along with multiple game modes and customization options.

## 🎮 Features

- ❤️ **Cute Theme** – Play using **Hearts** and **Stars** instead of Xs and Os  
- 👥 **Two Game Modes**
  - Player vs Player
  - Player vs Bot 🤖  
- 🧑‍🤝‍🧑 **Custom Player Names** – Enter your own names before starting  
- 🎨 **Choose Your Icon** – Select between ❤️ or ⭐  

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript** – Game logic
